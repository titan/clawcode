# `.claw/design/UI/`

Each **first-level subdirectory** is named after a **product, company, or design system** (e.g. `stripe`, `linear.app`, `vercel`). Folders hold **reference captures** (screenshots, exports) used when reasoning about layout, typography, or component patterns — similar to a mood board.

## Conventions

- **No README per brand** — content varies; this file documents the tree as a whole.
- Add new folders when you collect references for another brand; keep filenames descriptive inside each folder.

## Git

Image-heavy; prefer **not** committing huge dumps, or use **Git LFS** for large assets.
